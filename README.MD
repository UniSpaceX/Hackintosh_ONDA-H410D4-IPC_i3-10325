### bios 设置：
+ 关 `cfg lock`
+ 关 `csm`
+ 关 `quite boot`
+ 关 `vt-d`
+ 关 安全启动
+ 预显存 由 `32M` 改成 `64M`
+ apeture size 由 `256M` 改 `1024M`
+ DVMT Memory 改成 `Max`
+ 开 `Above 4GB decoding`
+ ...
> 其他的bios配置 看个人爱好。比如我改了功耗墙到 150W。内存的xmp没啥用，开不开也最高到2666.
